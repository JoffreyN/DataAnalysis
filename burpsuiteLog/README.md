# 分析burpsuite日志
将burpsuite日志按请求字段分类统计，保存为excel
## 说明
1. 运行脚本后，输入burpsuite日志文件名(脚本所在文件夹)
2. burpsuite日志内容格式见目录中burpsuiteLog.txt
3. burpsuite日志文件最好不要超过50M
4. 运行结果见目录中burpsuiteLog.xlsx
